# Todo List Senior

Application mobile de gestion de tâches développée avec **React Native** et **Expo SDK 54**.

> Projet développé par **Sam-s-dev**

---

## Fonctionnalités

- Ajout, suppression et modification du statut des tâches
- Recherche instantanée par titre
- Filtres : Toutes / Actives / Terminées
- Compteur de tâches restantes
- Persistance des données via AsyncStorage
- Animations légères (fade-in, fade-out, bounce)
- Design responsive (Android & iOS)

---

## Lancement rapide

```bash
cd todo-app
npm install
npx expo start
```

Scannez le QR code avec l'application **Expo Go** sur votre téléphone.
